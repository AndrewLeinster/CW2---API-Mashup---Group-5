About APIs

API stands for "Application Programming Interface", APIs are used in computing to access data or a library of methods that can be used by a piece of software or an operating system. They can also access web services. One of the most common uses of API's is to access data, many webssites make they're data accessible through APIs, for example, Facebook uses APIs to allow developers to access the data of users Facebook profiles. Most APIs can be accessed through programming languages such as java or python and some can be accessed using a web browser.
Our group has chosen to use a weather API that shows the weather forecast for a specific date and time in a city that someone wishes to travel to. We have integrated this API using Javascript and linked it to our webpage to show the information taken from an online source. The option to view more information also allows the user to see a slightly more detailed forecast changing from just showing the conditions and brief description of the weather to also show the sunrise and sunset times the temprature wind speed and expected precipitation levels.
On the Ducks page we have also linked our site to an API that shows pictures of... dogs? We did this to get practice pulling images from APIs.
